Android Guide

  • 1
    Uninstall Real Growtopia

    Jika Anda memiliki Growtopia asli di perangkat Anda, uninstall terlebih dahulu.

  • 2
    Install GROWRUSH APK

    Download dan install APK GROWRUSH yang telah kami sediakan.

    Download APK
  • 3
    Install App Pendukung

    Download salah satu aplikasi hosts berikut untuk koneksi:

    PowerTunnel Hosts Go Virtual Host
  • 4
    Buka GROWRUSH APK

    Buka aplikasi GROWRUSH dan gunakan akun GROWRUSH Anda untuk login.

Windows Guide

  • 1
    Buka Folder Hosts

    Tekan Win+R, lalu paste C:\Windows\System32\drivers\etc dan tekan Enter.

  • 2
    Edit File Hosts

    Buka file hosts dengan Notepad (Run as Administrator).

  • 3
    Tambahkan Host

    Tambahkan baris berikut ke akhir file hosts:

    5.196.204.227 growtopia1.com
    5.196.204.227 growtopia2.com
    5.196.204.227 www.growtopia1.com
    5.196.204.227 www.growtopia2.com
  • 4
    Simpan dan Launch

    Simpan file hosts dan buka Growtopia untuk bermain.

PowerTunnel (Android)

  • 1
    Install PowerTunnel APK

    Download dan install aplikasi PowerTunnel dari link di tab Android.

  • 2
    Konfigurasi Hosts

    Pergi ke Plugin → Hosts → Open the Setting, lalu masukkan URL:

    https://growtopia.id/android
  • 3
    Connect

    Kembali ke menu utama dan klik tombol "CONNECT", lalu buka Growtopia.

Hosts Go (Android)

  • 1
    Install Hosts Go

    Download dan install aplikasi Hosts Go (No Root) dari link di tab Android.

  • 2
    Download Hosts File

    Pergi ke Hosts Editor → enable → Download Hosts File, lalu masukkan URL:

    https://growtopia.id/android
  • 3
    Apply dan Start Protection

    Klik Apply, lalu start protection, dan buka Growtopia.

Virtual Host (Android)

  • 1
    Install Virtual Host

    Download dan install aplikasi Virtual Host dari link di tab Android.

  • 2
    Download Hosts File

    Download file hosts dari link berikut:

    Download Hosts File
  • 3
    Pilih Hosts File

    Buka aplikasi Virtual Host → SELECT HOSTS FILE, lalu pilih file hosts yang telah Anda download.

  • 4
    Buka Growtopia

    Setelah mengatur hosts file, buka Growtopia untuk bermain.

Mac Guide

  • 1
    Buka Folder Hosts

    Buka Finder → Go → Go to Folder, lalu masukkan /private/etc/hosts.

  • 2
    Edit File Hosts

    Copy file hosts ke Desktop, edit dengan TextEdit, lalu tambahkan baris IP yang sama seperti Windows.

  • 3
    Simpan dan Pindahkan Kembali

    Simpan file yang telah diedit dan pindahkan kembali ke folder aslinya.

  • 4
    Buka Growtopia

    Buka Growtopia untuk bermain di server kami.

Surge5 (iPhone)

  • 1
    Install Surge5

    Download dan install aplikasi Surge5 dari App Store.

  • 2
    Import Profile

    Buka Surge5 dan import profile dari URL berikut:

    https://growtopia.id/ios
  • 3
    Izinkan Konfigurasi VPN

    Izinkan konfigurasi VPN saat diminta oleh aplikasi.

  • 4
    Connect dan Buka Growtopia

    Klik tombol Connect, lalu buka Growtopia untuk bermain.
