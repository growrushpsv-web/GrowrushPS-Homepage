INSTALL GAME

Instalasi Android (Tanpa Root)

  • Install Hosts Go melalui browser atau PlayStore.
  • Pilih menu Hosts Editor dan masukkan konfigurasi manual.
  • Gunakan URL DNS berikut:
https://as.panel.growsoft.cc/android/1518

Instalasi Windows

Buka folder C:\Windows\System32\drivers\etc lalu edit file hosts menggunakan Notepad (Run as Admin).

Paste kode berikut di baris paling bawah:

15.235.227.241 growtopia1.com
15.235.227.241 growtopia2.com
