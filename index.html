Android
1
Uninstall Real Growtopia
Jika Anda memiliki Growtopia asli di perangkat Anda, uninstall terlebih dahulu.
2
Restart Device (Opsional)
Restart perangkat Anda untuk memastikan tidak ada cache yang tersisa.
3
Install GROWRUSH APK
Download dan install APK GROWRUSH yang telah kami sediakan.
Download PowerTunnel Download Hosts Go Download Virtual Host
4
Buka GROWRUSH APK
Buka aplikasi GROWRUSH dan gunakan akun GROWRUSH Anda untuk login.
Catatan: Pastikan Anda telah mengunduh salah satu dari aplikasi di atas untuk dapat terhubung ke server kami.
Windows
1
Buka Folder Hosts
Tekan Win+R, lalu paste C:\Windows\System32\drivers\etc dan tekan Enter.
2
Edit File Hosts
Buka file hosts dengan Notepad (Run as Administrator).
3
Tambahkan Host
Tambahkan baris berikut ke akhir file hosts:
15.235.227.241 growtopia1.com
15.235.227.241 growtopia2.com
15.235.227.241 www.growtopia1.com
15.235.227.241 www.growtopia2.com
4
Simpan dan Launch
Simpan file hosts dan buka Growtopia untuk bermain.
Catatan: Jika Anda mengalami kesulitan, pastikan Notepad dijalankan sebagai Administrator.
PowerTunnel (Android)
1
Install PowerTunnel APK
Download dan install aplikasi PowerTunnel dari link berikut:
Download PowerTunnel
2
Buka PowerTunnel
Buka aplikasi PowerTunnel yang telah Anda install.
3
Konfigurasi Hosts
Pergi ke Plugin ‚Üí Hosts ‚Üí Open the Setting, lalu masukkan URL berikut:
https://www.growrush.my.id/GrowRushPS.txt
4
Connect
Kembali ke menu utama dan klik tombol "CONNECT", lalu buka Growtopia.
Catatan: Pastikan PowerTunnel berjalan di latar belakang saat Anda bermain Growtopia.
Hosts Go (Android)
1
Install Hosts Go
Download dan install aplikasi Hosts Go (No Root) dari link berikut:
Download Hosts Go
2
Buka Hosts Go
Buka aplikasi Hosts Go yang telah Anda install.
3
Download Hosts File
Pergi ke Hosts Editor ‚Üí enable ‚Üí Download Hosts File, lalu masukkan URL berikut:
https://www.growrush.my.id/GrowRushPS.android
4
Apply dan Start Protection
Klik Apply, lalu start protection, dan buka Growtopia.
Catatan: Aplikasi ini tidak memerlukan akses root untuk berfungsi.
Virtual Host (Android)
1
Install Virtual Host
Download dan install aplikasi Virtual Host dari link berikut:
Download Virtual Host
2
Download Hosts File
Download file hosts dari link berikut:
Download Hosts File
3
Pilih Hosts File
Buka aplikasi Virtual Host ‚Üí SELECT HOSTS FILE, lalu pilih file hosts yang telah Anda download (GROWRUSH-HOST.txt).
4
Buka Growtopia
Setelah mengatur hosts file, buka Growtopia untuk bermain.
Catatan: Pastikan file hosts yang Anda download adalah versi terbaru untuk menghindari masalah koneksi.
Mac
1
Buka Folder Hosts
Buka Finder ‚Üí Go ‚Üí Go to Folder, lalu masukkan /private/etc/hosts.
2
Edit File Hosts
Copy file hosts ke Desktop, edit dengan TextEdit, lalu tambahkan baris berikut:
15.235.227.241 growtopia1.com
15.235.227.241 growtopia2.com
15.235.227.241 www.growtopia1.com
15.235.227.241 www.growtopia2.com
3
Simpan dan Pindahkan Kembali
Simpan file yang telah diedit dan pindahkan kembali ke folder aslinya.
4
Buka Growtopia
Buka Growtopia untuk bermain di server kami.
Catatan: Anda mungkin perlu memasukkan password administrator saat memindahkan file kembali ke folder aslinya.
Surge5 (iPhone)

Install Surge5

Download Surge5

Import Profile

Izinkan Konfigurasi VPN
Izinkan konfigurasi VPN saat diminta oleh aplikasi.
4
Connect dan Buka Growtopia
Klik tombol Connect, lalu buka Growtopia untuk bermain.
Catatan: Surge5 memerlukan iOS 13.0 atau versi lebih baru untuk berfungsi dengan baik.
