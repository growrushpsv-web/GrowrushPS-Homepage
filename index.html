Tutorial Android

  • Uninstall Real Growtopia

    Hapus aplikasi Growtopia asli jika ada di HP kamu.

  • Download App Pendukung
    Download PowerTunnel Download Hosts Go Download Virtual Host
  • Install & Mainkan

    Install APK GROWRUSH dan login menggunakan akun kamu.

Tutorial Windows

  • Buka Folder Hosts

    Tekan Win + R, lalu paste: C:\Windows\System32\drivers\etc

  • Edit File Hosts

    Buka file hosts dengan Notepad (Run as Administrator).

  • Tambahkan Host

    Tambahkan baris berikut di bagian paling bawah:

    5.196.204.227 growtopia1.com
    5.196.204.227 growtopia2.com
    5.196.204.227 www.growtopia1.com
    5.196.204.227 www.growtopia2.com
  • Simpan dan Mainkan

    Save file dan buka Growtopia.

PowerTunnel Setup

  • Install PowerTunnel

    Download & Install PowerTunnel APK via link Android.

  • Buka PowerTunnel

    Buka PowerTunnel app.

  • Konfigurasi Hosts

    Menu Plugin → Hosts → Open Setting. Masukkan URL:

    https://growtopia.id/android
  • Connect

    Kembali ke menu utama, klik CONNECT dan buka GT.

Hosts Go Setup

  • Install Hosts Go

    Download Hosts Go (No Root) dari link di tab Android.

  • Buka Hosts Editor

    Buka Aplikasi -> Hosts Editor.

  • Download Hosts File

    Masukkan URL:

    https://growtopia.id/android
  • Apply dan Start

    Klik Apply & Start Protection.

Virtual Host Setup

  • Install Virtual Host

    Download Virtual Host APK dari link di tab Android.

  • Download Hosts Manual

    Download file hosts manual dari link berikut:

    Download .txt
  • Pilih Hosts File

    Buka App -> Click "SELECT HOSTS FILE". Pilih file GROWRUSH-HOST (2).txt.

  • Connect

    Switch ON (Hijau) dan Login GT.

Mac Tutorial

  • Buka Folder Hosts

    Buka Finder -> Go -> Go to Folder, lalu ketik: /private/etc/hosts

  • Edit File Hosts

    Copy file hosts ke Desktop, edit dengan TextEdit.

  • Tambahkan Host

    Paste IP Server di paling bawah (sama seperti Windows).

  • Simpan dan Mainkan

    Save & Replace file asli di /private/etc/hosts. Buka Growtopia.

Surge 5 (iOS)

  • Install Surge 5

    Download Surge 5 di App Store.

  • Import Profile

    Buka Surge 5, pilih Import Profile. Masukkan URL: https://growtopia.id/ios

  • Izinkan VPN

    Izinkan Buat VPN Profile.

  • Connect & Mainkan

    Connect & Mainkan!
